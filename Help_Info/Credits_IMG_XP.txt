 Programs of IMG_XP package
 - VHD_XP_Create.exe - Make VHD file and Install XP in VHD by using Grub4dos Boot Menu on HDD
 - IMG_XP_Compact.exe - Make Portable and Mini XP Windows VHD Image files booting with Grub4dos Menu on HDD or USB
 - IMG_XP_Update.exe - to Update XP Image file after booting from full XP e.g. running from FiraDisk RAMDISK.
 - Win_Restore.exe - to Restore XP as OS on other Partition of HDD or USB-HDD drive using Windows VHD Image File as Source
 - USB_XP_Fix.exe - to Update XP HDD Image file on USB with tweaks and UsbBootWatcher including usb_xp_init.cmd
 - USB_W7_Fix.exe - to Update Win7 VHD Image file on USB with tweaks and UsbBootWatcher
 - VHD_W7_Compact.exe - to Copy Win7 VHD content to New VHD file of Reduced Size and make it suitable for Booting from USB-drive

 - BOOT_USB.exe can be used to make USB-drive bootable with Grub4dos Menu and using Boot Image files (VHD IMG or ISO)
 - BOOT_IMG.exe can be used to make Grub4dos Boot Menu for Boot Image files (VHD IMG or ISO) on existing bootable drives (HDD or USB)

 - VHD_W8_Compact.exe - to Copy Windows 8 folders and files to New VHD file of Reduced Size and using Full registry of Installed Windows 8
 - UEFI_MULTI.exe - Make Multi-Boot USB-Harddisk Booting with Boot Manager Menu and using Boot Image files (VHD IMG or ISO)
 - UFD_FORMAT.exe - to Format USB-stick with two partitions for Booting with Boot Manager Menu on BIOS or UEFI computer
 - USB_Part_Flip.exe - to change which primary partition of the USB-stick is visible in Windows
